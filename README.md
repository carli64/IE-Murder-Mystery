# IE Murder Mystery

## Description

A Python-based simulation of a Cluedo-style murder mystery game, set in a classroom environment.
The game uses custom characters (classmates and professors) and scenarios to recreate a logic-based mystery where the goal is to determine the culprit, location, and method.

## Features

* Cluedo-inspired gameplay logic
* Custom characters based on a classroom setting
* Scenario-based mystery simulation
* Python implementation for game logic and flow

## Installation

```bash
git clone https://github.com/carli64/IE-Murder-Mystery.git
cd IE-Murder-Mystery
```

## Usage

```bash
python campus_killer-15.py
```

## Requirements

* Python 3.9.6

## How It Works

The program simulates a mystery by:

* Selecting a random culprit, location, and method
* Allowing the player (or system) to deduce the correct combination
* Running logic similar to the classic Cluedo board game
